## Tap the list button in the top-left corner to open the file menu.

- `New canvas`: makes a fresh drawing.
- `Open`: opens a Grafite file with the `.gft` ending.
- `Import a file`: brings in an image on a new layer.
- `About`: Shows Grafite details, including version, credits, and release notes.
- `Help`: opens the handbook with shortcuts, tips, and support resources
- `Quit`: closes Grafite.
