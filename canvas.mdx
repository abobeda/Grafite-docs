## The Canvas button changes the paper and the artboard.

### Background choices:

- `White`
- `Off white`
- `Paper: Soft`
- `Paper: Hard`
- `Paper: Textured`

### Artboard controls:

- `W` and `H`: set width and height.
- `Swap`: flips width and height.
- `Infinite`: turns on infinite canvas mode.
- `Presets`: opens ready-made canvas sizes.

### Preset groups:

- `Screen`: 1024x768, 1280x800, 1600x1000, 1920x1080, 2048x1536
- `A Series`: A2, A3, A4, A5
- `US Paper`: 8.5 x 11 in, 11 x 17 in
- `Social`: Instagram Story, Instagram Post 1:1, Instagram Post 3:4, X Post Portrait, X Post Landscape

<Tip>
  Good to know:

  - When you choose a finite artboard, Grafite resets the view so the artboard fits nicely.
  - If you use touch on the width or height fields, a short hold opens editing mode.
  - After the number box opens, you can type the new size and press `Enter`.
</Tip>
