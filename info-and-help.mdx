# Info and Help

The **Info and help** button opens documentation links and app information.

1. The **Info and help** button opens help links and app information.
2. **Grafite Help** opens this documentation.
3. **Journal** opens the Grafite blog.
4. **Report an Issue** starts a support email.
5. **About Grafite** shows the current app version and copyright information.
