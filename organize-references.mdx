---
title: "Organize References"
---

Use **Organize references** when you want to clean up, rearrange, or manage the images inside a board.

A board is not just a folder of images. You can shape it into a visual sequence, move stronger references closer together, remove images that no longer fit, and prepare the board for reviewing a direction.

<Frame>
  ![Organize References Board](/images/organize-references-board.jpg)
</Frame>

## Enter organize mode

Click **Organize references** in the top right of the Board screen.

When organize mode is active, Grafite changes the board into a selection and arrangement mode. This lets you manage references without opening each image one by one.

## Select references

Click a reference to select it.

You can select one or multiple references before choosing an action. Selected references are visually marked so you can clearly see what will be affected.

Use selection when you want to:

1. Move several references to another board.
2. Remove multiple references from the current board.
3. Review a smaller group inside a larger board.

## Drag and drop to reorder

You can drag references to rearrange them inside the board.

This is useful when you want to control the visual flow of a moodboard. For example, you might place broad inspiration first, then details, colors, materials, poses, lighting, or final direction references later.

To reorder references:

1. Enter **Organize references** mode.
2. Drag a reference image.
3. Drop it into a new position.
4. Grafite saves the new order for that board.

The order is local to the board, so the same reference can still appear differently in another board.

## Move references to boards

You can send selected references to another board.

This helps when one image fits more than one idea, or when you want to split a large board into smaller directions.

<Frame>
  ![Add Board](/images/add-board.jpg)
</Frame>

Depending on the action you choose, you can use this to:

1. Add selected references to an **existing board**.
2. **Create a new board** from selected references.
3. Keep the references in the current board.
4. Move them out of the current board if you no longer want them there.

<Frame>
  ![Add New Board](/images/add-new-board.jpg)
</Frame>

## Remove references

You can remove selected references from the board.

Use this to clean up images that no longer match the direction of the board.

Removing a reference from a board does not always mean the image disappears everywhere. If that reference is used in another board, it can remain available there. If it only exists in this board, removing it may remove it from your Grafite library.

## Finish organizing

When you are done, click **Done** to leave organize mode.

Your board returns to its regular browsing mode, where clicking a reference opens it in Focus View.

## Tips

1. Put the strongest references near the beginning of a board.
2. Group similar images together so the board is easier to read.
3. Place detail shots near the larger images they support.
4. Use separate boards when one moodboard starts becoming several different ideas.
5. Remove weak or repeated references so the board stays clear.
